Kako radi

        

          
Svaki dan HPB šalje izvod kao email s XML + PDF attachmentima. Knjigovodstvo
          traži da im oba forwardaš.

          
Ovo sučelje ti:

          
    
            
  • Pokaže sve izvode koji su u IZVODI CO 2026/
    • 
            
  • Po statusu: ✓ poslan (s datumom) · ◐ nije poslan
    • 
            
  • Jednim klikom otvori Mail.app compose s XML+PDF već attached
    • 
          

          
Bulk akcija: "Pošalji sve neposlane" → otvara po jedan Mail.app compose za svaki. Ti samo potvrdiš Send.

      Auto-detekcija po fajlu:
      
    
        
  • 🏦 Bank izvodi (camt.053 XML pattern <accountNumber>.YYYY###.xml/.pdf) → instant save u bank folder (bez Claude-a, free)
    • 
        
  • 📄 PDF računi → Claude Vision parsa → match s placeholder-om → save u <invoice-folder>/Placanje MMM YY/Supplier/
    • 
      

      PDF račun match logic:
      
    
        
  1. Iznos ±0.05 € · datum ±10 dana · supplier similarity
    2. 
        
  2. Score ≥ 0.6 = MATCH (uses placeholder folder)
    3. 
        
  3. Inače → suggested folder iz parsed PDF-a (svejedno spremljen)
    4. 
      

      Cost: ~$0.01 po PDF-u (Claude Sonnet 4.5). XML/PDF izvodi su besplatni.
